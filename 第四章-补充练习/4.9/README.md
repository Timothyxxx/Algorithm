#### 问题描述
&emsp;&emsp;求3个字符串的最长公共子序列,即$LCS(A_{1,n},B_{1,m},C_{1,p})$;也可扩展到更多子字符串。
#### 问题分析
&emsp;&emsp;首先类似于2个的情况，如果这三个串的最后一位相同，那么这三个串的最长公共子序列就是它们各自去掉最后一个字符的三个串的结果+1(**这里的优化子结构易证**);否则,这三个串中至少有一个串的最后一位毫无意义，因此去掉这位，再进行比较即可。注意：显然不能判断是不是有两个串的末尾都无意义，或者和某两个的LCS有任何关系。
&emsp;&emsp;因此递归方程如下:
$$LCS(A_{1,n},B_{1,m},C_{1,p})=
\begin{cases}
R_{ij}+1& \text{$A_n = B_m = C_p$} \\
max(LCS(A_{1,n-1},B_{1,m},C_{1,p}),LCS(A_{1,n},B_{1,m-1},C_{1,p}),LCS(A_{1,n},B_{1,m},C_{1,p-1}))& \text{其他}
\end{cases}
$$
